1.Challenges in Retail（小売業の挑戦）


>- **Multi-Channel…実店舗やネット販売など複数の販売形式を用いて顧客との接点を多様化させる販売戦略**
>- **Omni-Channel…販売形式にとらわれずに同じサービスを提供することができる販売戦略**
 
>- **Serial-Process…連続した処理（実店舗は実店舗で、ECサイトはECサイトで別々に売る）**
>- **Interconnected…相互接続する（実店舗とECサイトを共用することで片方の弱点を補うことができる）**

```
*EC…インターネットを使用した商売全般のこと
*POS…ネットワークを利用した販売情報を管理するシステム
*SCM…製造から販売までの一連の流れを効率化する経営手法
*IMS…データベースを階層的に利用したデータ管理システム
*WMS…倉庫内の在庫管理や入出荷などの倉庫管理システム
```

```
*Online…コンピュータがネットワークで接続されていてサービスを受けられる状態
*Offline…コンピュータがネットワークで接続されていない状態
*Consumer…消費者
*Enterprise…企業
``` 

```
*Intranet…企業内でネットワークがつながっていて、情報が共有できる状態
*Internet…世界規模のネットワークで世界各国の個人や組織がつながっている
*Centralized…情報を一か所に集めること
*Distributed…情報を分散させて共有すること
*Batch…一定の時刻にデータをまとめて処理すること
*Realtime…即時にデータを処理すること
*Monolithic…一つに統制されたプログラムをつくること。変更されるごとにアプリケーション全体を再構築しなければならない
*Microservice…一つ一つ細かくプログラムを作ることで他のプログラムに影響がでないようにする
```

2.Microservice Design（小さなサービスを一つにまとめたアプリケーションの構成）

```
*API…アプリケーションプログラミングインターフェイス。アプリケーションの機能を利用するためのもの。

*CDN…大容量のデジタルコンテンツをインターネット上で大量配信することができる。アクセスが集中したときなどにサーバがダウンしにくくなる。
*API Gateway…AWSのサービス。開発者がAPIを作成したり管理したりすることができる
*AWS…アマゾンのクラウドサービス
*Router…ネットワークとネットワークを接続する機器
*DNS Failover…webアプリケーション、webサーバなどの正常性とパフォーマンスを監視するもの
 ```

3.Global Operation（海外戦略）

```
*Network Latency…ネットワークの遅延
　*Physical Distance（欧米）…物理的な距離
　*Great Firewall（中国）…中国国外からのアクセス制限
*Regional Regulations…地域の法律
　*Privacy Law（欧、中）…個人情報保護法
```

```
*Authority Delegation…権限の委託
*Standardization & Automation…標準化&自動化
```
